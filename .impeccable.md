## Design Context

### Users
This is a **showcase/presentation page** for demonstrating a multi-layered AI agent system architecture inspired by the Blue Archive (碧蓝档案) game universe. The audience includes developers, tech enthusiasts, and Blue Archive fans who will view it as an impressive visual presentation of the system design. The page needs to immediately communicate "this is something special" — it's not a working dashboard, it's an architectural vision rendered beautifully.

### Brand Personality
**Playful, Polished, Cinematic**

The interface should feel like stepping into the Blue Archive world itself — bright, lively, and full of character, but with the sophistication of a high-end product presentation. It evokes a sense of wonder, pride in the system's complexity, and the warmth of the Blue Archive universe.

### Aesthetic Direction

**Visual Tone**: Blue Archive game UI meets cinematic web storytelling. Think of it as "what if SCHALE had a real command interface that was also gorgeous."

**Core Visual Language of Blue Archive**:
- **Color Palette**: Dominated by sky blue (#128AFA / bright cerulean) and white. Each school has its own accent color scheme:
  - Abydos: warm sand/desert gold
  - Gehenna: fiery red/crimson
  - Trinity: royal blue/gold
  - Millennium: teal/cyan/tech green
  - Hyakkiyako: deep purple/traditional Japanese
  - Shanhaijing: jade green/vermillion
- **Signature Elements**: Halo (光环) — floating geometric shapes above characters, a core visual identity. Cross/geometric motifs. Rounded, friendly UI elements with subtle depth.
- **Typography**: Mix of clean sans-serif (for UI elements) with stylized display fonts. Japanese game UI aesthetic — clean, readable, with personality in headers.
- **Light & Bright**: The game is fundamentally bright and optimistic. NOT dark/gloomy. Sky backgrounds, soft clouds, warm light.
- **School Emblems**: Each school has distinctive logo/emblem designs — geometric, symbolic, high-contrast.

**Anti-References**:
- Generic AI dashboard aesthetic (cyan-on-dark, gradient text, glassmorphism everywhere)
- Boring corporate architecture diagrams
- Anything that looks like "an AI made this" — no cookie-cutter cards, no identical grids

**References for Web Execution**:
- Scroll-driven storytelling with staggered reveals
- Cinematic hero sections with depth and motion
- Interactive 3D-like elements reacting to cursor
- Exploratory layouts that invite discovery rather than passive reading
- Illustrated/hand-crafted feel mixed with clean UI — the page should feel like it was designed by a human with taste

### Design Principles

1. **World-Building First**: Every visual choice should reinforce the Blue Archive universe. Use school colors, halo motifs, emblem-style graphics. The page IS the SCHALE system, not just a diagram of it.

2. **Cinematic Storytelling**: Don't dump all information at once. Use scroll-driven narrative to reveal the hierarchy layer by layer — Sensei at top, then SCHALE, then schools unfold, then clubs within schools. Each section transition should feel like a scene change.

3. **Motion with Purpose**: Animations should be abundant but meaningful — elements float like halos, connections pulse like data flowing, schools appear with character. Use staggered reveals, parallax depth, and hover-reactive elements. No bounce/elastic easing — smooth exponential deceleration only.

4. **De-AI-ified Craft**: Break symmetry intentionally. Vary card sizes. Use hand-crafted SVG illustrations for school emblems. Mix typography scales dramatically. Add imperfections and personality that prove a human designed this. No identical repeating patterns.

5. **Bright & Alive**: Keep the palette bright and warm like the game. Sky blue dominates with school-specific accents adding variety. Use generous white space, soft shadows, and warm light effects. The page should feel like a sunny day at Kivotos.

### Tech Stack
- Single HTML file, no framework dependencies
- CSS animations + keyframes for motion (no JS animation libraries needed for v1)
- SVG for custom illustrations and connection lines
- Google Fonts for typography
- Responsive but desktop-first (this is a presentation piece)
